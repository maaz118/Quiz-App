# 🎯 Quiz App
This is a simple and interactive Quiz Application developed using **HTML, CSS, and JavaScript**. The app allows users to attempt multiple-choice questions and view their final score at the end.

---
## 🚀 Features
- 🧠 Multiple-choice quiz questions
- ✅ Instant answer selection
- 📊 Score calculation at the end
- 🔄 Restart quiz option
- 💾 Data handling using JavaScript
- 📱 Responsive design for different devices
---

## 🛠️ Technologies Used
- **HTML** – Structure of the app  
- **CSS** – Styling and layout design  
- **JavaScript** – Logic and functionality
